What I Learned

I learned the basic concept of an SSG. I learned that they are a super fast way to get a simple static site up and going. And they are scalable and can grow if you code it out right.

My Experience With Harp

Harp in particular is super easy to setup. It also allows you to use several different languages like Jade, EJS, LESS, SASS, Markup, etc.. without having to worry about setup. It just complies into HTML, CSS, JS no matter which you use. The ability to use a layout with the yield variable and partials is super useful. Using global and meta data was interesting as well. EJS feels like it's trying to accomplish something similar to react but on a much lighter scale. I had trouble getting the Google Maps API to work, got my own key and everything, but no cigar.

Future Work With Harp

I could use Harp to make simple blogs, info sites, brochures, potfolio sites, landing padges etc for myself or clients.
